CARPETA: Logo del Hostal Rivera
================================
Coloca aquí el logo del hostal.

Nombre recomendado : logo.png  o  logo.svg
Tamaño recomendado : 300 x 100 px  (o proporcional)
Formatos aceptados : PNG (fondo transparente), SVG, WEBP

Ejemplo de uso en una plantilla Django:
  {% load static %}
  <img src="{% static 'images/logo/logo.png' %}" alt="Logo Hostal Rivera">
