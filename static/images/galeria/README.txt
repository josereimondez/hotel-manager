CARPETA: Galería de fotos del Hostal
=====================================
Coloca aquí fotos generales del hostal: interior, comedor, recepción, entorno, etc.

Nombres sugeridos:
  - interior-comedor.jpg
  - recepcion.jpg
  - salon.jpg
  - entorno-ancares.jpg
  - fachada-noche.jpg

Tamaño recomendado : 800 x 600 px  (galería)  /  1200 x 800 px  (destacadas)
Formatos aceptados : JPG, WEBP

Ejemplo de uso en una plantilla Django:
  {% load static %}
  <img src="{% static 'images/galeria/comedor.jpg' %}" alt="Comedor Hostal Rivera" class="img-fluid">
